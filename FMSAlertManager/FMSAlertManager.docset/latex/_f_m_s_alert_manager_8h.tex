A block that is called when the alert view's button is pressed. 

{\ttfamily Response\-Block}s are simply blocks that don't take any arguments and don't return any values. {\ttfamily \hyperlink{interface_f_m_s_alert_manager}{F\-M\-S\-Alert\-Manager}} uses {\ttfamily Response\-Block}'s to define the actions that will be triggered when the corresponding {\ttfamily U\-I\-Alert\-View} button is pressed.
